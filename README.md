# CSVインポートをRakeタスクで行いました。　　

### ・```rails c```でコンソールを立ち上げずにCSVのインポートができるように実装。  　

### ・インポート中にエラーが発生した場合、それを知らせる機能も実装。　　

### (作動方法)  
コマンドで作動させるディレクトリに移動した後、　　

```rake import_csv:users```　　

上記のrakeを作動させることでCSVデータをインポート。　　

その後```User.all```とするとインポートされたデータを確認可能。
